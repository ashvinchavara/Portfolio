AI-Native CRM & Learning Platform

Mrivan AI - School CRM & AI Tutor

A comprehensive school management and learning system. It incorporates an Express.js backend connecting to PostgreSQL (Supabase) and leverages Google Gemini for step-by-step empathetic pedagogical tutoring, complete with automatically synchronized chat history, homework verification, and CBT mock tests.

  • AI Tutor (Mr. Ivan AI): Empathetic learning assistant that doesn't just give answers, but guides students step-by-step.
  • CRM Portal: School setup, classroom/section definition, homework assignment/submission, and student tracking.
  • Smart Content: Custom markdown study guide builder, automated quiz generator, and score evaluators.

A client-server framework. The Flutter app (web/mobile) communicates with a custom Node.js/Express server hosted on Render. User profiles and metadata are synced dynamically from Supabase Auth to a public profiles table, securing CRM resources using Row Level Security (RLS).

BLE Mesh & Scheduler Infrastructure

Presence Tracker - BLE Mesh Proximity Attendance

A modern attendance and scheduling portal. Designed around an infinite department hierarchy (College -> Course -> Year -> Class) and a 3NF normalized MySQL database backend. The attendance mechanism uses Bluetooth Low Energy (BLE) Mesh network topologies, using the teacher device as the root and student devices as leaves.

  • Infinite Hierarchy Model: Department and layout modeling that dynamically links classes, courses, and timetables.
  • Proximity Presence: Secure verification of attendance using BLE mesh protocols without requiring internet connectivity for the client.
  • Normalized Timetable: 3NF structured database schemas that map schedules, recurring classes, and temporary teacher swaps.

Uses a customized advertising payload where teacher devices act as roots, sending synchronized session keys. Leaf nodes scan, verify, sign with device-bound credentials, and propagate verification payloads back through the BLE mesh network to establish a tamper-proof attendance grid.

Flutter Node.js Express.js MySQL (3NF) BLE Mesh API Vanilla HTML/CSS/JS (Admin Dashboard)
Repository
Native Android Client Utility

Adtendo - Android Proximity Attendance Client

The native Android application supporting the Adtendo presence ecosystem. Implements dedicated user flows for Students, Teachers, and Heads of Departments, reinforced by secure device biometrics, foreground location matching, background BLE scanning/advertising mechanisms, and customized home screen widgets.

  • Biometric Cryptography: Integrates biometric authentication utilizing custom helper classes to decrypt server-provided user keys.
  • Home Screen Widget: Custom app widget updates that display real-time attendance stats directly on the device home screen.
  • Role-Based Navigation: Dynamically loaded dashboards mapping distinct options for students, teachers, and admins.

Leverages background services to handle location updates, Bluetooth permissions, network connectivity tracking, and boot completions. Restarts services automatically upon device reboot using Broadcast Receivers to maintain proximity awareness.
